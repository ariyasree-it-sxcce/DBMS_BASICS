     -----------------------------------------------------------------------------------------------
     > DDL Commands ---> Data Definition Language
        > Used to define,modify and delete database structures like table,column,schemas
            1. Create
            2. Alter 
            3. Drop
            4. Truncate
            
             ----> 1. Create
                      Used to create a new schema, create a new table, or define columns and their  data types.

                 Syntax1: create schema schema_name
                 Example: create schema dummy

                 Syntax2: create table table_name(
                            var_name1 data_type,
                            var_name2 data_type,
                          )
                 Example: create table teacher(
                            Name varchar(100),
                            Age int,
                            Degree varchar(100)
                          )
             ----> 2. Alter
                       Used to modify an existing table.
                       
                 Syntax1: alter table table_name add column var_name_to_be_added data_type
                 Example: alter table teacher add column Department varchar(100)

                 Syntax2: alter table table_name change var_name_to_be_changed new_name data_type
                 Example: alter table teacher change Degree Deg varchar(100)

             ----> 3. Drop
                       Used to delete a table completely from the database.
              
                 Syntax:  drop table table_to_be_deleted
                 Example: drop table teacher
            
             ----> 4. Truncate 
                        Used to delete all data inside a table, but keeps the table structure unchanged.

                 Syntax:  truncate table table_name
                 Example: truncate table teacher 

     -----------------------------------------------------------------------------------------------
     >DML Commands ---> Data Manipulation Language
                   ---> insert,update,deleting data

                   >INSERT
                    Syntax:  INSERT INTO table_name VALUES(DATA1,DATA2,DATA3)
                    Example: INSERT INTO student VALUES("ARIYA",19,"IT")

                   >UPDATE
                    Syntax1:  UPDATE table_name SET variable_name=new_value
                    Example:  UPDATE student SET Age=23
                               > It updates all old value to new value

                    Syntax2:  UPDATE table_name SET variable_name=new_value WHERE variable_name=value_to_be_updated
                    Example1: UPDATE student SET Age=18 WHERE Age=19
                    Example2: UPDATE student SET Age=18 WHERE Name="John"
                               > It update only the specified position to be updated 

                   >DELETE
                    Syntax1:  DELETE FROM table_name
                    Example1: DELETE FROM student 
                               > It delete all data from student table 

                    Syntax2:  DELETE FROM table_name WHERE variable_name=value
                    Example2: DELETE FROM student WHERE Name="ARIYA"

     >DQL Commands ---> Data Query Language
                   ---> use select command for retrieve data and view

                   Example1: SELECT * FROM student 
                             >Display all contents of Table.
                             > '*' specifies all.
                    
                   Example2: SELECT Name FROM student
                             >Display only the Names from the table.
                    
                   Example3: SELECT Name,Age FROM student
                             >Display Name as well as Age from the table.
